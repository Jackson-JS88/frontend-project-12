# Чат на React (frontend-project-12)

Чат в реальном времени на React с бэкенд-сервером.

## 📝 О проекте

Чат в реальном времени с возможностью:
- Регистрации и авторизации пользователей
- Создания, переименования и удаления каналов
- Отправки сообщений в реальном времени
- Фильтрации нецензурных слов
- Интернационализации (русский/английский)

## 🌐 Деплой
Проект доступен по ссылке: [https://frontend-project-12-0d85.onrender.com](https://frontend-project-12-0d85.onrender.com)

## 📋 Требования
- Node.js (версия 18 или выше)
- npm (устанавливается вместе с Node.js)
